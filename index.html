Keliling Segitiga

Rumus Keliling Segitiga yaitu :

K = S1 + S2 + S3

Dimana :

K = Keliling

S1 = Sisi A

S2 = Sisi B

S3 = sisi C

Luas Segitiga

Rumus Luas Segitiga yaitu :

L = 1/2 * a * t

Dimana :

L = Luas

a = alas segitiga

t = tinggi segitiga
